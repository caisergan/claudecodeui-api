Claude Session API Demo

This is a minimal app built on top of the external CloudCLI API. It uses API-key-authenticated project discovery, Claude session listing, normalized message history, and session resume via POST /api/agent.

Open API docs →
Status: Enter an API key, then load projects.
Flow: projects → Claude sessions → history → resume session
